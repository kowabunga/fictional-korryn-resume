Projects

Blog Page

This project started out as a simple blog page to learn how to use CSS Grid and make it responsive at various screen sizes. I also had a chance to play around with using Sass to write my CSS for it, as it makes it much cleaner when writing and adds rules that I had not considered (rules such as -ms- and -webkit- for older browsers).

Blog Administrator Page

This project is an expansion of the Blog Page project I mentioned previously. Having had a look at Bootstrap, I thought it would be good to learn how to make a more complex webpage in a simpler manner. Bootstrap enables the creation of complex webpages without the hassle that comes with having to manually decide everything with pure CSS.

Making Meals

This restaurant website was made as an experiment into adding JavaScript and JQuery to a typical HTML/CSS website I created earlier on in my learning experience. I had a lot of fun doing this, though it could be frustrating at times when the scripts wouldn't do as I expected, though eventually I figured them out.
